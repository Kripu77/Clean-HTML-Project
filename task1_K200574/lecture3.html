A Quick Review of JavaScript - a client side scripting

Revision on fundamental concepts of JavaScript

What is Javascript

This week's lecture started with a general introduction of JavaScript a widely used client side scripting language and introduced by Netscape in 1996. The lecturer explained about the latest version of JS which happens to be the ES6 which was introduced in 2015 and brought some new features with the introduction of concepts such as let, const which has block scope and arrow function which are typically used in writing functions in an more cleaner way and effective way.

The lecturer explained about various important concepts such as how to declare and intiliaze variable in JavaScript , which is by either using Var, Let, or Const keyword. Furthermore, the lecturer also explained various forms of Loops such as While Loop, Do while Loop and For Loop which can be applied to make our program more efficient. It was intresting to know how JavaScript is used to manipulate the Document Object Model and make the site respond according to user activites.

Additionally, the lecturer explained how the forms with multiple inputs works and communicate with our data servers with various requests. The lecturer also explained about various libraries of JavaScript available such as React.js, Vue.jS, Angular.js. It was very fascinating to know how the use of JavaScript has gotten a bigger grasp on the server side scripting with the introduction of Node.js and simplified the process in learning.
