Introduction to Server-Side Programming

This week's lecture introduced to the wider world of Server Side Scripting Using PHP.

Server-side Development

The lecturer begun the lecture with the explanation about what server side development is, which bascially refers to ideas to generate dynamic content using programming language such as PHP, ASP.NET, Node,js, etc. the most fascinating stuff about various langauages available is they do one thing in common which is to generate HTML, CSS and JavaScript despite being dynamically different than each other.

The lecturer explained that similar to JavaScript, PHP is also a dynamically typed language and the major factor to know was the syntax for loops, conditionals and operators are almost identical to JavaScript. The concept is only a bit different when we reach to the point of function, classes and while defining variable which is by using a "$" sign while declaring.

The lecturer also described that the PHP files have an extension of .php and the opening and closing tag which is abit different than any other languages I have encountered so far. The nature of PHP and how it is written is a bit of jargon when the concpets are quite unclear.
